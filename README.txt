A simple game written in python3, frozen into an windows exe using cx_Freeze.
You can play the game by just running the "BubbleBusterCode.py" script. The other files are for turning it into an executable. 
The goal is to move around your "submarine" and pop as many bubbles as possible to get the highest score before time runs out!
Also, getting a certain amount of points makes go to the next level!
Use the arrow keys to move. Press escape or backspace to quit the game.
It's currently in version 0.1.
